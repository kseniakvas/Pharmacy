# 🏥 Домашня Аптечка

## Опис
Це консольний застосунок для зручного обліку домашньої аптечки. Програма дозволяє відстежувати наявність ліків, контролювати їхні терміни придатності, керувати залишками та вчасно отримувати нагадування про прострочені препарати. Завдяки зручним звітам та кольоровому оформленню ви завжди знатимете, що саме потрібно докупити та що варто викинути.

---

## Технології

- .NET 8
- Entity Framework Core (Code First)
- SQLite
- LINQ (для побудови звітів)

---

## Функціонал

Застосунок працює через зручне текстове меню в циклі та підтримує такі можливості:

1. **Додати ліки** — збереження назви, типу (Таблетки/Сироп/Мазь/Інше), кількості, терміну придатності та категорії.
2. **Переглянути аптечку** — вивід таблицею (назва, тип, кількість, термін придатності). Прострочені ліки позначаються червоним кольором, а ті, що закінчуються — жовтим.
3. **Пошук** — швидкий пошук за частиною назви препарату або його категорією.
4. **Використати ліки** — зменшення кількості обраного препарату (з вбудованим захистом від від'ємних значень).
5. **Поповнити запас** — збільшення кількості та оновлення терміну придатності.
6. **Видалити ліки** — видалення запису за ID з обов'язковим підтвердженням дії користувачем.
7. **Звіти** — перегляд прострочених ліків, ліків що закінчуються (термін < 30 днів), та ліків що закінчилися (кількість 0).

> ⭐ **Бонус — Список покупок:** автоматичний перелік ліків які треба купити (кількість 0 або прострочені).

## Структура проєкту

```
HomePharmacy/
├── HomePharmacy.Data/          # Робота з базою даних
│   ├── Models/
│   │   ├── Medicine.cs         # Модель ліків
│   │   ├── Category.cs         # Модель категорії
│   │   └── MedicineType.cs     # Enum типів ліків
│   ├── AppDbContext.cs         # Конфігурація EF Core
│   └── Migrations/             # Міграції бази даних
└── HomePharmacy.Console/       # Консольний застосунок
    ├── Medkit.cs               # Основна логіка та меню
    ├── InputValidations.cs     # Валідація вводу
    └── Program.cs              # Точка входу
```

## Запуск програми

**1.** Переконайтеся що встановлено [.NET 8 SDK](https://dotnet.microsoft.com/download)

**2.** Клонуйте репозиторій:
```bash
git clone [посилання на репозиторій]
```

**3.** Перейдіть у папку проєкту:
```bash
cd HomePharmacy
```

**4.** Запустіть програму:
```bash
dotnet run --project HomePharmacy.Console
```

> 💡 База даних SQLite створюється автоматично під час першого запуску.
